## Instalar Angular
npm install -g @angular/cli

## Criar o projeto do site
npx ng new portal

## Entrar na pasta do projeto
cd portal

## Inicializar o servidor angular (modo dev)
npx ng serve

## Criar um novo componente
npx ng g c header

## Criar um novo componente de rodapé
npx ng g c footer

## Para atualizar as dependências
npx npm install

## Criar um novo componente de local
npx ng g c local

## Criar um novo componente de regional
npx ng g c regional

## Criar um novo componente de internacional
npx ng g c internacional

## Criar um novo componente de home
npx ng g c home
